# Integration: Webhooks for Notifications

**TaskRun** allows you to send automatic notifications to messaging tools like **Discord**, **Microsoft Teams**, and **Google Chat** using **webhooks**. This is useful for alerting about successful tasks, errors, retries, or simply for continuous monitoring.
