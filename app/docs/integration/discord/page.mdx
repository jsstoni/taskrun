# Discord

### How to Get the Webhook

1. Open Discord and go to the channel where you want to receive notifications.
2. Go to "Edit Channel" → **Integrations** → **Webhooks**.
3. Create a new webhook and copy the URL.

### Message Format

TaskRun sends messages like this:

```
✅ Successful Task: Weather API Request
🕒 Executed at: 12:00
⏱ Duration: 124 ms
```

In case of an error:

```
❌ Task Error: Weather API Request
🕒 Executed at: 12:00
💥 Error: Timeout
```
