**TaskRun** is a cron-based task scheduler that allows you to schedule and execute external API calls automatically and periodically. It is designed to simplify the automation of processes that rely on external services, enabling you to define, control, and monitor the execution of scheduled tasks with flexibility and precision.

---

## Key Features

* **Cron-based scheduling**: Define tasks using standard cron syntax.
* **External API execution**: Make HTTP requests to RESTful APIs or any configurable endpoint.
* **Error handling**: Configurable retries and notifications in case of failures.
* **Monitoring and logs**: Detailed logging of each execution, whether successful or failed.
* **Dynamic parameters**: Allows sending variable data with each execution.
